# MercApp

## Instrucciones de uso

### 1. Clonar el repositorio

git clone https://github.com/KleberTipantiza/MercApp.git

## 2. Archivo `seed.json`

El archivo `mercapp-api/seed.json` contiene los productos y categorías iniciales que se cargan al iniciar el backend. Se puede modificar este archivo para personalizar el catálogo.

## 3. Instalar dependencias e Iniciar Servidores 

Para el Backend (Express) se debe ejecutar los siguientes comandos en la terminal

cd mercapp-api

npm install

npm run dev (Esto inicia el servidor en http://localhost:3000.)

Para el Frontend (Vue) se debe ejecutar los siguientes comandos, dentro de otro terminal

cd vue-project

npm install

npm run dev (Esto inicia la SPA en http://localhost:5173.)

## 3. Recomendaciones

Se recomienda usar Visual Studio Code como entorno de desarrollo.
